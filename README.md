## 🚀 快速开始

### 环境要求

- Node.js >= 14.0
- npm / yarn / pnpm
- 现代浏览器（Chrome、Firefox、Safari、Edge）

### 运行项目

每个项目都是独立的，进入对应目录后：

```bash
# 进入项目目录
cd [项目文件夹]/project

# 安装依赖
npm install

# 开发模式
npm run dev

# 生产构建
npm run build
```

### 查看文章

每个项目文件夹都包含 `博客.md` 文件，详细说明了项目的实现原理和技术细节。

```bash
# 查看项目文章
cat [项目文件夹]/博客.md
```

## 📖 技术栈总览

### 前端框架
- **Vue 2/3** - 渐进式 JavaScript 框架
- **React** - Facebook 的 UI 库（部分项目）

### 3D 图形
- **Three.js** - WebGL 3D 库
- **Babylon.js** - 另一个 3D 引擎选项

### 构建工具
- **Vite** - 下一代前端构建工具
- **Webpack** - 模块打包器

### UI 组件库
- **Element Plus** - Vue 3 组件库
- **Element UI** - Vue 2 组件库

### 桌面应用
- **Electron** - 跨平台桌面应用框架

### 后端/服务
- **Node.js** - JavaScript 运行时
- **Express** - Web 框架
- **WebSocket** - 实时通信

### 其他库
- **GSAP** - 动画库
- **fabric.js** - Canvas 库
- **MediaPipe** - 机器学习框架
