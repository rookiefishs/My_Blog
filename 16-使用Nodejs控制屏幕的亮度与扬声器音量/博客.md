# 使用Nodejs控制屏幕的亮度与扬声器音量

## 1. Nodejs如何控制音量
1. 首先下载**louddness**第三方库,这个库提供了**Nodejs**音量控制的API
2. 获取当前系统音量API
```js
// 导入Nodejs音量控制API
const loudness = require('loudness');

// 获取当前的音量
loudness.getVolume().then(volume => {
  console.log(`当前系统音量为:${volume}%`);
});
```
3. 设置指定系统音量API
```js
// 导入Nodejs音量控制API
const loudness = require('loudness');

// 设置系统音量
loudness.setVolume(50).then(() => {
  console.log(`音量已设置为50%`)
});
```

## 2. Nodejs如何控制屏幕亮度
1. 首先下载**brightness**第三方库,这个库提供了**Nodejs**控制屏幕亮度的API
2. 获取当前的屏幕亮度
```js
// nodejs操作系统亮度第三方库
const brightness = require('brightness')

// 获取当前的屏幕亮度
brightness.get().then(light => {
  console.log(`当前的屏幕亮度为:${(light * 100).toFixed(2)}%`);
})
```
3. 设置指定的屏幕亮度
```js
// nodejs操作系统亮度第三方库
const brightness = require('brightness')

// 设置当前的屏幕亮度
brightness.set(1).then(res => {
  console.log('屏幕亮度设置成功!')
})
```
